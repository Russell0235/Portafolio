# APLICACIÓN

El programa implementa una tabla de símbolos usando una tabla hash con listas enlazadas para resolver colisiones. Esta tabla se utiliza en compiladores para guardar y consultar información sobre los identificadores (variables y funciones) dentro de un programa. Mediante un menu interactivo puedes agregar, buscar o eliminar identificadores al elegir la opcion de agregar te pedira la clave(nombre del identificador), su tipo (int, float, entre otros), su valor, su ámbito(global o local) y la línea en donde se guardara ese identificador en la tabla de simbolos (0-9).
